Opens and parses an Ogre3D .scene file exported from Blender in order to recreate the scene content into a SceneNode.

Parameters:
_sceneFileThe .scene file to load.
_localSceneMgrThe SceneManager instance on which the main SceneNode will be attached to, and on which entities and lights will be created.
_setEntitiesDefines whether the entities found in the file are created or not.
_setLightsDefine whether the lights found in the file are created or not.
_setCameraDefine whether the cameras found in the file are created or not (NOT CURRENTLY ACTIVE).
_offsetXDefine the X offset to use when placing the created nodes.
_offsetYDefine the Y offset to use when placing the created nodes.
_offsetZDefine the Z offset to use when placing the created nodes.
