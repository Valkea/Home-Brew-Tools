Detailed Description

This class returns a singleton with a unique identifier for each registered event type.

At the first EventType::get call, the EventType is created and stored, thereafter this same instance is returned each time the EventType is called using get static method. 

            hbt::EventType& onClockUpdate = hbt::EventType::get("CLOCK_UPDATE");
            
            hbt::Signal< void(ClockEvent) > onClockEvent;
            onClockEvent( hbt::ClockEvent( onClockUpdate, 0,0,0,0,0 ) ); // dispatch using predefined EventType
            onClockEvent( hbt::ClockEvent( hbt::EventType::get("CLOCK_STOP"), 0,0,0,0,0 ) ); // dispatch using EventType defined on the fly
            
            hbt::EventType::freeAll();
Warning:
The copy constructor and the assignment operator are privates to avoid any EventType copies... thus we are 100% sure that all EventTypes using the same name refer to the same instance and that this instance will not disappear because it was passed as a temporary argument.
