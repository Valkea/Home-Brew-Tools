Detailed Description

template<class... Args>
struct hbt::IDelegate< Args >

This abstract class aggregates method, functions and functor pointers under a unique type.

Class member method pointers, functor pointers or classical function pointers can be registered using inheritors of this abstract class (MethodDelegate, FunctionDelegate and FunctorDelegate), whatever they are virtual, static and or const as long as they are waiting for the same arguments list (same order, same types, same number)...

Template Parameters:
ArgsThe argument list of all MethodDelegate, FunctionDelegate and FunctorDelegate using this IDelegate.
